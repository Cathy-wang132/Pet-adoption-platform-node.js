# 宠物领养平台-node.js(管理端+用户端）
### 1、介绍
本宠物领养平台是一个前后端分离的Web应用系统，采用经典的B/S架构设计。系统包含三大核心模块：后端API服务、用户端前台和管理端后台。后端基于Node.js和Express框架构建RESTful API，前端采用Vue2结合Element UI组件库实现界面交互，数据库选用MySQL存储业务数据。整个项目涵盖宠物信息管理、会员体系、领养流程、资质认证、权限管理等完整业务功能！
